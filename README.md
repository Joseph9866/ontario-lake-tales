# Ontario's Lakes Turtles Website

A responsive, accessible educational website about Ontario's endangered turtle species, their habitats, and conservation efforts.

## Project Structure

- **Home Page (index.html)**: Introduction to Ontario's turtles with featured species and calls to action
- **Endangered Turtles Section**: Overview and individual pages for 8 turtle species
- **Conservation Projects (solutions/saving-turtles.html)**: Information about turtle conservation initiatives
- **Contact Page (communication/contact.html)**: Form for reporting sightings and getting involved
- **Additional Pages**: Lakes, Parks, Research, and Scientific information pages

## Features

- Fully responsive design (mobile, tablet, desktop)
- WCAG 2.1 AA accessibility compliance
- Semantic HTML5 structure
- CSS3 with variables for consistent theming
- CSS-only lightbox for image viewing
- CSS accordion for expandable content
- Accessible form with HTML5 validation
- "Back to top" functionality
- Breadcrumb navigation

## Color Palette

- Primary: #1C352D (Deep Lake Blue)
- Secondary: #84994F (Forest Green)
- Accent/CTA: #FFC72C (Sunburst Yellow)
- Background: #F5F5DC (Soft Beige)
- Text: #333333 (Rich Charcoal)

## Technical Requirements Met

- No frameworks or libraries (pure HTML5/CSS3)
- Responsive design across all devices
- Semantic HTML elements
- CSS variables for consistent theming
- CSS-only interactive elements (lightbox, accordion)
- Accessible navigation and forms
- Optimized for performance and SEO
- Valid HTML and CSS
